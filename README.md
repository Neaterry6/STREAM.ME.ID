# StreamMe Music 2026 🎵
 
StreamMe Music is a modern, responsive web app that lets users **search, stream, and download music and videos** directly from their browser. Designed with simplicity, speed, and usability in mind, StreamMe provides a seamless experience for discovering trending songs, albums, playlists, and artists.
  
## Features
 
 
- 🔍 **Search Music & Videos** – Find tracks, albums, playlists, or artists.
 
- 🎧 **Play & Stream** – Listen to music or watch videos directly in the app.
 
- ⬇️ **Download** – Download audio or video files easily.
 
- 💾 **History & Logs** – Track your recent searches, plays, and downloads.
 
- 🌐 **Trending & Recommendations** – Automatically loads trending content on launch.
 
- 🎨 **Beautiful Responsive UI** – Works perfectly on web and mobile devices.
 
- 💖 **Custom Loading Screen & Logo** – StreamMe branding with a sleek animation.
 

  
## Tech Stack
 
 
- **Frontend:** HTML, CSS, JavaScript (with responsive design)
 
- **Backend:** Node.js, Express.js
 
- **APIs:** QasimDev APIs for YouTube, Spotify, and MusicBrainz content
 
- **History Tracking:** `history.js` for storing user search & playback history
 

  
## Installation
 
 
1. Clone the repository:
 

 `git clone https://github.com/Neaterry6/STREAM.ME.ID.git cd streamme-music ` 
 
1. Install dependencies:
 

 `npm install ` 
 
1. Start the server:
 

 `npm start ` 
 
1. Open your browser and go to:
 

 `http://localhost:3000 `
